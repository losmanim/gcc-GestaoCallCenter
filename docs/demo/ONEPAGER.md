# GCC — Gestão Call Center
## One-Pager Comercial

---

### O que é?
Plataforma web completa para call centers de **telecomunicações** (NOS, MEO, Vodafone). Integra site de leads, toolkit de vendas, CRM e aprovação digital.

### Para quem?
- Call centers de vendas de telecom
- Equipas internas de retenção e fidelização
- Revendedores e agentes autorizados

### Porquê o GCC?

| Problema | Solução GCC |
|----------|-------------|
| Sem script padronizado | Prancha + Script + Objeções guiadas |
| Dados perdidos em Excel | CRM com IBAN, CVP, contratos |
| Aprovação demora dias | Cliente aprova por email em segundos |
| Gestão às cegas | Dashboard com KPIs em tempo real |
| Leads desperdiçados | Badge de notificação + conversão 1 clique |
