# Guia de Teste Grátis — Passo a Passo

## Como conduzir um trial de 7 dias do início ao fim

---

## Antes de começar: o que precisas ter pronto

### Checklist técnica

- [ ] Domínio próprio (ex: gcc.seudominio.pt) — ~5€/ano
- [ ] Alojamento dos ficheiros (Netlify, Vercel, ou hosting) — 0-3€/mês
- [ ] Email comercial configurado (ex: comercial@gcc.pt)
- [ ] Link de demonstração gravado (Loom ou similar) para enviar
- [ ] Template de email de boas-vindas
- [ ] Número de WhatsApp/Business dedicado

### Checklist de vendas

- [ ] Proposta comercial preenchida com dados do cliente
- [ ] Contrato de teste (pode ser um email simples)
- [ ] Calendário com 3 marcos: D1, D3, D7
- [ ] Perguntas para fazer no onboarding

---

## Passo 1 — O cliente diz "sim" ao teste

**Isto acontece durante a reunião de demonstração ou num follow-up.**

### O que fazer imediatamente:

```
1. "Ótimo! Vou preparar o acesso e envio-lhe o link ainda hoje."
2. Enviar email de boas-vindas (ver template abaixo)
3. Adicionar ao calendário: reunião de onboarding (D1)
```

> **Nunca:** "OK, depois mando qualquer coisa." — Define já a data do onboarding.

### Template de email de boas-vindas:

```
Assunto: ✅ GCC — O teu teste grátis de 7 dias está pronto

Olá [NOME],

Obrigado por aceitares experimentar o GCC.

O teu teste grátis de 7 dias começa hoje.

🔗 Link de acesso: https://gcc.seudominio.pt/gestao
👤 Login: admin@[email-do-cliente].pt
🔑 Senha: [senha temporária]

📅 Onboarding: [DATA] às [HORA] — duração 30 min
   Link: [meet/Zoom/WhatsApp]

O que vamos fazer no onboarding:
1. Visão geral do sistema (10 min)
2. Configurar os primeiros clientes e serviços (10 min)
3. Integrar o site de leads (5 min)
4. Responder a dúvidas (5 min)

Qualquer coisa, estou no WhatsApp: [TEU NÚMERO]

Resposta rápida,
[TEU NOME]
GCC — Gestão Call Center
```

---

## Passo 2 — Onboarding (Dia 1, 30-45 min)

### Preparação antes da chamada

- [ ] Ter o sistema aberto e logado
- [ ] Ter dados de exemplo prontos (ou usar os que já vêm)
- [ ] Ter o site de propaganda aberto noutro separador
- [ ] Ter o telemóvel com o link de aprovação pronto
- [ ] Fechar o microfone de fundo e notificações

### Script do onboarding

| Min | O que mostrar | Como dizer |
|-----|---------------|------------|
| 0-3 | Apresentação | "Vou mostrar-te o essencial em 30 min. No final, vais conseguir criar clientes, contratos e gerir os leads." |
| 3-8 | Site propaganda | "Este é o site público. Os clientes preenchem o formulário e os leads aparecem automaticamente aqui." |
| 8-12 | Dashboard | "Aqui vês: total de clientes, contratos ativos, receita mensal, leads novos. Num relance sabes como está o negócio." |
| 12-18 | Clientes + Serviços | "Vou mostrar-te como criar um cliente e um serviço. Presta atenção aos campos IBAN e CVP — são específicos das telecom." |
| 18-25 | Contratos + Aprovação | "Esta é a parte mais fixe. Crias um contrato, envias para aprovação, e o cliente aprova do telemóvel. Vou mostrar ao vivo." |
| 25-30 | Leads + Dúvidas | "Os leads do site aparecem aqui. Com 1 clique convertes em cliente. Agora é a tua vez — queres experimentar?" |

### No final do onboarding:

```
"Muito bem! Agora tens 7 dias para explorar.
No dia 3 faço-te um check rápido por WhatsApp.
No dia 7 vemos os resultados e decides se queres continuar.

Alguma pergunta antes de eu desligar?"
```

### Tarefas pós-onboarding (tu):
- [ ] Enviar resumo por email com links e login
- [ ] Agendar check do Dia 3 (WhatsApp, 5 min)
- [ ] Agendar reunião do Dia 7 (15 min)

---

## Passo 3 — Check-in (Dia 3, 5 min por WhatsApp)

### Objetivo:
- Saber se está a usar
- Tirar dúvidas
- Reforçar valor

### Template:

```
Olá [NOME]! 👋
Como está a correr o teste do GCC?

Já conseguiu:
✅ Criar alguns clientes?
✅ Enviar algum contrato para aprovação?
✅ Ver leads a chegar do site?

Se precisar de alguma coisa é só dizer.
Se ainda não teve tempo, sugiro começar pelos Contratos — é onde o sistema faz mais diferença.
```

### Possíveis respostas e como reagir:

| Resposta | O que fazer |
|----------|-------------|
| "Ainda não tive tempo" | "Sem problema. Quer marcar 15min amanhã para fazermos juntos o primeiro contrato?" |
| "Estou a adorar" | "Boa! Lembra-te que no dia 7 fazemos o fecho. Já vou preparando a proposta." |
| "Tenho dúvidas em X" | Responder ou oferecer 5 min de videochamada |
| "Não estou a conseguir Y" | "Posso ligar agora 5 min e mostrar?" |
| Não responde em 24h | "Só um toque — quer que remarcamos o onboarding? Posso ajudar a configurar os primeiros dados." |

---

## Passo 4 — Fecho (Dia 7, 15 min)

### Preparação:
- [ ] Revisitar os dados que o cliente criou durante o teste
- [ ] Preparar proposta final com o plano certo
- [ ] Decidir se dás desconto de lançamento

### Script de fecho:

| Momento | O que dizer |
|---------|-------------|
| Abertura | "Então, como foram os 7 dias? O que achaste?" |
| Medir valor | "Quantos contratos conseguiste criar? Os leads estão a chegar bem?" |
| Apontar valor | "Viste como a aprovação por email acelera o fecho? Já imaginaste a tua equipa toda a usar isto?" |
| Proposta | "Baseado no que viste, recomendo o plano [Pro/Enterprise] para [N] gestores. Fica [VALOR]€/mês." |
| Fecho | "Queres avançar? Posso ativar já hoje." |

### Se disser sim:

```
"Perfeito! Vou enviar a referência multibanco para pagamento.
Assim que confirmar, ativo o acesso definitivo.
Bem-vindo a bordo!"
```

### Tarefas pós-fecho:
- [ ] Enviar referência multibanco ou link de pagamento
- [ ] Atualizar senha do cliente (se aplicável)
- [ ] Enviar contrato comercial (se aplicável)
- [ ] Agendar reunião mensal de acompanhamento

### Se disser "vou pensar":

```
"Claro! O que precisas de pensar mais?
Se for o preço, posso oferecer [desconto de lançamento].
Se for funcionalidades, posso mostrar como [X] funciona."

[Se insistir em pensar:]
"Compreendo. Vou deixar o teste ativo por mais 3 dias para
poderes mostrar à equipa. Depois disso, se não avançarmos,
desativo. Combinado?"
```

### Se disser "não":

```
"Sem problema. Posso saber o que não funcionou?
É para melhorar o sistema."

[Anotar o feedback!]
"Obrigado pela honestidade. Se mudarem de ideias,
o link continua disponível para reativar quando quiserem."
```

## Passo 5 — Pós-venda (Dia 8+)

### Cliente que fechou:

| Quando | Ação |
|--------|------|
| Semana 1 | "Como está a correr a primeira semana?" |
| Mês 1 | "Já tens métricas para comparar antes/depois?" |
| Mês 2 | "Queres adicionar mais gestores?" |
| Mês 3 | Pedir testemunho para usares como prova social |
| Semestral | Renovação anual com desconto |

### Cliente que não fechou:

| Quando | Ação |
|--------|------|
| 1 mês | "A oferta de lançamento ainda está disponível se quiserem reativar." |
| 3 meses | "Novas funcionalidades: [lista 2-3]. Continuam interessados?" |
| 6 meses | Newsletter com dicas e métricas do sector |

---

## Opções para oferecer no fecho (para aumentar conversão)

| Opção | O que dizes | Efeito |
|-------|-------------|--------|
| **Desconto lançamento** | "Para os primeiros clientes, ofereço 50% no primeiro mês." | Reduz objeção de preço |
| **Mês grátis** | "Se fechares hoje, o primeiro mês é por minha conta." | Cria urgência |
| **Referral** | "Para cada cliente que indicares, tens 1 mês grátis." | Gera novas leads |
| **Preço trancado** | "Este preço é para clientes do lançamento. Nunca mais vai ser tão barato." | FOMO |
| **Setup grátis** | "Normalmente cobro 149€ de setup. Para vocês é grátis." | Aumenta valor percebido |

---

> **Regra de ouro:** Quem usa o sistema durante os 7 dias, compra. Quem não usa, não compra. O teu trabalho no onboarding e check-in é garantir que ele usa.
